# 🍪 Cookie Sales Dashboard | Power BI

## Overview
The Cookie Sales Dashboard is an interactive Power BI report designed to monitor and analyze cookie sales performance. It provides a clear overview of revenue, shipment trends, and customer sales data, enabling users to identify patterns, evaluate performance, and support data-driven decision-making.

## Dashboard Features
- Interactive sales dashboard built in Power BI
- Customer sales table with customer ID, name, country, and cookies shipped
- Revenue KPI card displaying total revenue
- Time-series visualization showing cookies shipped by order date
- Year-based sales analysis
- Easy-to-read layout for business reporting

## Key Insights
- Track total revenue generated from cookie sales.
- Monitor the number of cookies shipped over time.
- Compare customer orders across different countries.
- Identify sales trends throughout the year.
- Analyze customer purchasing activity using detailed transaction data.
